Features
What's included

Everything you need to evaluate AI agents on real coding tasks, from the loop to the metrics.

↻ Agent Loop

Plan, Act, Observe, Reflect. Automatic tool dispatch with configurable step limits and early stopping.

☰ 8 Metrics

Pass rate, tool efficiency, cost tracking, context utilization, error recovery, steps, and duration.

▭ 4 Memory Strategies

Sliding window, summarization, RAG, and hybrid. Switch with one config line.

⚙ Tool Execution

Bash, file read/write, search. Sandboxed with timeouts and error capture.

▨ Multi-Agent

Planner, Executor, Reviewer. Orchestrated revision loops with role-based coordination.

▸ CI Integration

GitHub Actions workflow. Auto-diagnoses failing PRs and posts agent results as comments.

How it works
Four steps, zero intervention

Define a task, let the agent work, get scored results.

01

Define

Write a YAML task with the bug description and test commands.

02

Plan

Agent reads the code, identifies the issue, and builds an action plan.

03

Execute

Uses tools to read, write, and test files autonomously.

04

Evaluate

Metrics computed, results logged, performance benchmarked.
