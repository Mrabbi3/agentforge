# Contributing to AgentForge

Thank you for your interest in contributing to AgentForge!

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/agentforge.git`
3. Install dev dependencies: `pip install -e ".[dev]"`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

- Run tests: `pytest`
- Type check: `mypy agentforge`
- Lint: `ruff check agentforge`
- Format: `ruff format agentforge`

## Pull Request Guidelines

- Write tests for new features
- Update documentation as needed
- Follow existing code style
- Keep PRs focused on a single change

## Areas for Contribution

- New memory strategies
- Additional tools
- Task definitions (coding and knowledge)
- Evaluation metrics
- Dashboard features
- Documentation improvements
